开发环境：
        keil uVision5

开发目的：
        利用STM32F105恒速采集ADXL345三轴加速度传感器数据，当ADXL345受到撞击时，采样速率提高。

硬件资源：
        1，STM32F105
        2，ADXL345三轴加速度传感器
        3，SPI接线方式
 
项目结果：
        低速下，MCU会以1.25ms/次的速率采集ADXL345的数据，利用滞后滤波降噪后，计算加速度值，当    发现加速度值大于1.5g时，提高采样速率至0.3ms/次。当MCU收到0xE5的数据时，会将加速度值发送给    串口。
	GPS收到卫星信号时，解析数据，发给串口。
